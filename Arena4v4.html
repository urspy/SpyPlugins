Overview

Arena4v4 is a production-ready, fast-paced team-based dueling plugin designed for Minecraft servers looking to offer ranked 4v4 matches with kit support, ELO ranking, arena management, and smart queueing—all packed in a lightweight and optimized JAR.

Support: For help, updates, or support, reach out via Discord or donate.

Critical Issues Fixed

1. NullPointerException Error – RESOLVED

  • Fixed initialization order: ConfigManager now loads before ArenaManager
  • Comprehensive null safety checks added across managers
  • Fallback configuration systems now in place

2. Configuration Loading – ENHANCED

  • Graceful error handling for missing or broken config files
  • Automatic fallback creation for default configurations
  • Verbose logging system added for easier debugging

3. WorldEdit Dependency – OPTIMIZED

  • WorldEdit made completely optional (softdepend)
  • Plugin detects and adapts if WorldEdit is not installed

4. Default Systems – IMPLEMENTED

  • Auto-loaded default kits: Iron and Diamond
  • Arena data files generate automatically if missing
  • Handles empty or missing configurations without crashing

Commands

Player Commands

Command Description
/4v4 join [kit] Join the matchmaking queue (optional kit name)
/4v4 leave Leave the current queue
/4v4 stats [player] View player statistics
/4v4 top [wins|kills|elo] View top players
/4v4 spectate [player] Spectate an ongoing match
/4v4 gui Open visual kit selector with one-click queue joining
/4v4 toggle Toggle match requests on/off

Admin Commands

Command Description
/4v4admin create <arena> Create a new arena
/4v4admin delete <arena> Delete an existing arena
/4v4admin setspawn <arena> <team> <pos> Set team spawn point in an arena
/4v4admin savekit <name> Save your current inventory as a kit
/4v4admin loadkit <name> Load a previously saved kit into inventory
/4v4admin setpos1 <arena> Set arena region position 1
/4v4admin setpos2 <arena> Set arena region position 2
/4v4admin saveblocks <arena> Save arena blocks for auto-reset
/4v4admin setlobby Set lobby location

Core Systems

Automatic Queue

Balanced 4v4 matchmaking system that ensures fair teams based on ELO ratings.

ELO Rating

Player ranking system saved via SQLite with persistent statistics tracking.

Spectator Mode

Live match viewing for non-players with first-person spectating options.

Kit System

Supports default & custom kits that can be saved and loaded by admins.

Arena Management

Create, delete, and configure multiple arenas with team spawn points.

SQLite Storage

Persistent statistics and match data stored in efficient SQLite database.

Installation Guide

  1. Drop arena4v4-1.0.0.jar into your server's /plugins/ folder.
  2. (Optional) Add WorldEdit if you prefer region-based arena setup.
  3. Start your server to auto-generate configs and default kits.
  4. Use /4v4admin commands to configure arenas and spawns.
  5. Ready to play!

Configuration Files

config.yml

Main plugin settings including:

  • Queue settings
  • ELO calculation parameters
  • Arena defaults
  • Database configuration

kits.yml

Predefined and saved kits including:

  • Default Iron and Diamond kits
  • Custom kit configurations
  • Kit permissions

Both files are auto-generated and editable after first server launch.

Tips for Admins

Multiple Arenas

Want different arenas per map? Use /4v4admin setspawn on multiple arenas to create variety.

Kit Balance

Balance kits for fairness—start with iron and diamond as base kits, then expand with custom variations.

Spectator Mode

Use spectator mode to monitor players or host live events with commentary.
